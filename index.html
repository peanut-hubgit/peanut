PeanutOS

Primeiro OS em Python

PeanutOS é um projeto educacional desenvolvido em Python que simula conceitos de um sistema operacional moderno.

Este projeto ainda está em desenvolvimento.

PeanutOS não é um sistema operacional real. É um projeto educacional criado para aprendizado, experimentação e demonstração de conceitos.

Recursos

Tudo o que você precisa para aprender

Uma base sólida em Python aplicada a conceitos clássicos de sistemas operacionais — organizada, legível e didática.

Interface Moderna

Layout limpo inspirado em desktops contemporâneos, com janelas, dock e barra de tarefas organizados em componentes reutilizáveis.

Desenvolvido em Python

Construído majoritariamente em Python, priorizando clareza, legibilidade e facilidade de leitura para quem está começando.

Projeto Educacional

Foco em aprendizado, experimentação e demonstração. Cada módulo é documentado para servir como material de estudo.

Em Evolução

Novos recursos, aplicativos internos e melhorias de interface estão em desenvolvimento ativo a cada iteração.
